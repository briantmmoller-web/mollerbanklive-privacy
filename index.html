Privacy Policy — MollerBankLive

MollerBankLive is a personal-use application built by Brian Moller to retrieve his own bank account and transaction data from Sydbank via the Enable Banking API (an open banking / PSD2 account information service), for the sole purpose of importing that data into a personal Excel spreadsheet for private financial tracking.

Data collected: Account balances and transaction history, accessed only for the account holder's own bank account(s) under explicit PSD2 consent granted through the account holder's bank.

Data storage: Retrieved data is stored locally on the account holder's own computer (in Excel files) and is not transmitted to, stored on, or shared with any third party, server, or service other than the direct API calls to Enable Banking required to retrieve it.

Data sharing: No data is sold, shared, or disclosed to any third party.
